# Rebel SDK

This repository contains the Rebel Software Development Kit (SDK) for [Rebel Engine](https://github.com/RebelToolbox/RebelEngine).

The Rebel SDK allows you to easily create external C++ libraries that can be linked to your Rebel Engine game.
You can extend existing classes, create your own classes and interface with other third-party libraries.
All you need to do is link your code with the Rebel SDK library and define three methods.

The Rebel SDK is available for all platforms supported by Rebel Engine:
- Linux
- macOS
- Windows
- Android
- iOS
- Web
